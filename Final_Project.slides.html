Transcript Data: What matters?¶

Field Name Description
Comments Number of comments on the TedTalk
Description A short blurb about the content of the TedTalk
Duration The duration of the talk in seconds
Event The Ted or TedX event that the talk took place at
Film Date The timestamp of the filming of the talk
Languages The number of languages in which the talk is available
Main Speaker The name of the main speaker in the talk
Name The official name of the TedTalk
Number of Speakers The number of speakers in the talk
Published Date UNIX TimeStamp for when the talk was published on Ted.com
Ratings A stringfield dicitionary of the various ratings given to the talk
Related Talks A list of dictionaries of recommended talks to watch next
Speaker Occupation The occupation of the main speaker
Tags The themes associated with the talk
Title Title of the talk
URL URL of the talk
